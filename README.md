# Fishable Today

Fishable Today is an Idaho-first fly fishing conditions dashboard.

The MVP helps anglers decide which local rivers and reaches are worth fishing today based on flows, weather, water temperature, hatch windows, and report signals.

## Current MVP

This first version is a static prototype deployed through GitHub Pages.

## Future Plan

- Add USGS streamflow data
- Add National Weather Service forecasts
- Add daily generated condition snapshots
- Add river profile pages
